== Description ==

Reconewsは投稿されたコンテンツの内容を解析して、関連する外部のニュースやコンテンツを取得して投稿ページに表示することで内容の充実を図ることができます。

= 外部ニュースは下記のいずれかのサイトより取得することができます = 
1. bing (bingの検索結果が表示されます。)
2. Googleニュース (Googleニュースの検索結果が表示されます。)

= Localization =
* Japan (default)

== Installation ==

= 管理画面上でインストールする場合 =
1. WordPress管理画面のメニュー「プラグイン -> 新規追加」をクリック
2. 検索窓で「Reconews」と検索して「今すぐインストール」ボタンをクリック
3. ボタンが「有効化」に変化した後、もう一度クリック

= ダウンロードしたフォルダをアップロードする場合 =
1. reconews-wpフォルダを'/wp-content/plugins/'ディレクトリにアップロードしてください。
2. 「プラグイン」メニューの「Reconews」を有効化してください。
